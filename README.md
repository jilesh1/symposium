# Symposium 🎓

"Symposium" is our collaborative web project built to master professional development. We honed our skills in Git version control, pull requests, and modular CSS/HTML design. Beyond coding, it was a test of teamwork, responsive architecture, and efficient communication—turning individual effort into a cohesive, functional web experience.



## 🚀 Key Features

* **Collaborative Workflow:** Managed development through GitHub feature branches and pull requests to ensure a clean codebase.
* **Dynamic CSS Slider:** Custom-built drag-to-scroll slider logic to create an app-like experience.
* **Grid & Flex Layouts:** Architected using modern CSS Grid and Flexbox for precise, scalable positioning.
* **Responsive Design:** Fully adaptive interface using media queries to ensure a seamless experience from mobile to desktop.



## 🛠 Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **HTML5** | Structure and semantic content |
| **CSS3** | Styling and layout design |
| **SCSS** | Modular, maintainable styles |
| **JavaScript** | Interaction logic (slider & responsiveness) |
